## Require
- Node
- npm
- Git

## Usage

- `git clone https://github.com/SettingDust/webpack-tampermonkey.git`
- `npm i`
- `npm run dev`
- Copy the content of `./test/header.js` to the TamperMonkey script editor.
- Open a webpage.

## Features

-   Test without `Ctrl+V`
-   Build a public version easily.
